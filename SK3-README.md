SK 3 — Simple Navigation Demo

Files:

- sk3-navigation.html
- sk3-styles.css

How to run:

1. Open `sk3-navigation.html` in your browser (double-click or use Live Server).
2. Use the top nav to jump between sections.

What it demonstrates:

- Internal hyperlinks (`#about`, `#services`, etc.)
- Basic responsive navigation with a small-screen toggle
- Clear, accessible anchor links and simple CSS styling
